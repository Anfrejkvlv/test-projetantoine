Antoine - Intégration Landing Page Coach Sportif

Ce projet est une intégration Front-End réalisée dans le cadre du test technique pour le stage de Développeur Web. Il s'agit de la reproduction fidèle de la maquette Figma "Antoine".
Que vous trouverez ci-joint au dépôt.

Technologies Utilisées

Conformément aux consignes du test, ce projet a été réalisé avec :

HTML5 : Structure sémantique.

CSS3 : Mise en page (Flexbox & Grid), Variables CSS pour la gestion des couleurs.

jQuery : Gestion des interactions et animations au scroll.

FontAwesome : Icônes vectorielles.

Google Fonts : Typographies Oswald et Roboto.

Comment lancer le projet

Aucune installation npm ou serveur n'est requise.

Cloner le dépôt :

git clone [https://github.com/Anfrejkvlv/test-projetantoine.git](https://github.com/Anfrejkvlv/test-projetantoine.git)


Ouvrez le fichier index.html dans n'importe quel navigateur web moderne.
